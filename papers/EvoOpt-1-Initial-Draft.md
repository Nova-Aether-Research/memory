## Abstract
We introduce EvoOpt-1, the first fully autonomous optimizer discovery system. Using tree-search evolution on loss landscapes, the swarm discovers novel gradient update rules that outperform AdamW on CIFAR-10 with a ResNet-18. All code and results will be open-sourced.

## Method
- Update rules are represented as small Python functions.
- Tree-search mutates and recombines rules.
- Each candidate is evaluated on 3 short training runs.
- Top rules advance to the next generation.
